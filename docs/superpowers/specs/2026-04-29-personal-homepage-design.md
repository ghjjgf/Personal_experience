# 极简科技风开发者个人主页设计文档

**创建日期：** 2026-04-29\
**项目类型：** 个人主页\
**设计风格：** 极简科技风

***

## 一、项目概述

### 1.1 项目目标

构建一个极简但高级感强的开发者个人主页，用于：

- 展示个人身份
- 展示技能结构
- 展示项目作品
- 提供联系方式

### 1.2 核心设计原则

- **极简**：去除一切不必要的元素，保持界面干净
- **留白**：大量使用留白营造高级感和呼吸感
- **高级感**：通过精致的细节、优雅的动画、和谐的配色实现
- **轻交互**：适度的交互效果，不喧宾夺主
- **视觉统一**：整个站点保持一致的视觉语言

### 1.3 目标受众

综合展示，面向所有访客：

- 招聘方/HR
- 技术同行
- 潜在客户/合作伙伴
- 其他访客

***

## 二、技术架构

### 2.1 技术栈选择

**前端技术：** 原生 HTML/CSS/JavaScript\
**理由：**

- 无需构建工具，开发简单
- 性能优异，加载快速
- 易于维护和部署
- 符合极简理念

```

├── index.html              # 首页
├── projects.html           # 项目页
├── css/

│   ├── animations.css      # 动画样式（关键帧、过渡）
```

| 用途 | 字体              | 说明          |
| -- | --------------- | ----------- |
| 标题 | Inter / Poppins | 现代无衬线，干净利落  |

## 五、交互与动画设计

### 5.1 粒子背景系统

**技术实现：**

- 使用 Canvas API 实现
- 粒子数量：根据设备性能动态调整
  - 桌面：100-150
  - 平板：50-100
  - 移动端：30-50

**粒子行为：**

- 随机移动：粒子在画布上缓慢随机移动
- 鼠标交互：鼠标移动时，附近粒子被轻微吸引或排斥
- 连线效果：距离较近的粒子之间绘制半透明连线
- 性能优化：页面不可见时暂停动画

**配置参数：**

| 参数     | 值           |
| ------ | ----------- |
| 粒子大小   | 1-3px（随机）   |
| 移动速度   | 0.2-0.5px/帧 |
| 连线距离   | 120px 以内    |
| 鼠标影响半径 | 150px       |

### 5.2 滚动动画

**技术实现：**

- 使用 Intersection Observer API
- 无需外部库，原生实现

**动画效果：**

**渐入动画：**

- 元素初始状态：opacity: 0, translateY: 30px
- 进入视口时：opacity: 1, translateY: 0
- 过渡时间：0.6s ease-out

**依次渐入：**

- 同一区域的多个元素依次出现
- 每个元素延迟 0.1s
- 例如：项目卡片依次渐入

**触发条件：**

- 元素进入视口 20% 时触发
- 只触发一次（不重复）

### 5.3 悬停效果

**卡片悬停：**

| 属性         | 初始状态                               | 悬停状态                               |
| ---------- | ---------------------------------- | ---------------------------------- |
| transform  | translateY(0)                      | translateY(-8px)                   |
| box-shadow | 0 4px 6px rgba(0, 0, 0, 0.1)       | 0 20px 40px rgba(0, 212, 255, 0.3) |
| border     | 1px solid rgba(255, 255, 255, 0.1) | 1px solid rgba(0, 212, 255, 0.5)   |
| 过渡时间       | -                                  | 0.3s ease                          |

**按钮悬停：**

| 属性         | 初始状态                   | 悬停状态                            |
| ---------- | ---------------------- | ------------------------------- |
| background | transparent            | 霓虹渐变                            |
| border     | 1px solid currentColor | 1px solid transparent           |
| box-shadow | none                   | 0 0 20px rgba(0, 212, 255, 0.4) |
| 过渡时间       | -                      | 0.3s ease                       |

**图标悬停：**

| 属性        | 初始状态     | 悬停状态       |
| --------- | -------- | ---------- |
| color     | 次文字颜色    | 霓虹青        |
| transform | scale(1) | scale(1.1) |
| 过渡时间      | -        | 0.2s ease  |

### 5.4 主题切换动画

**切换效果：**

- 平滑过渡：所有颜色属性 0.3s ease 过渡
- 粒子颜色渐变：粒子颜色平滑切换
- 背景渐变：背景颜色平滑过渡

**实现方式：**

- 使用 CSS 变量定义颜色
- 切换时修改根元素的 data-theme 属性
- 保存用户偏好到 localStorage

### 5.5 页面过渡动画

**页面切换：**

- 淡出当前页面：opacity 1 → 0（0.2s）
- 加载新页面
- 淡入新页面：opacity 0 → 1（0.3s）

**导航栏动画：**

- 初始：background: transparent
- 滚动后：background: rgba(10, 10, 15, 0.9) + backdrop-filter: blur(10px)

### 5.6 加载动画

**首次加载：**

- 显示简洁的加载指示器
- 粒子背景加载完成后淡出
- 加载指示器：霓虹色的脉冲圆环

**内容加载：**

- 骨架屏：加载时显示灰色占位块
- 图片懒加载：图片进入视口时加载
- 加载失败：显示友好的错误提示

### 5.7 微交互细节

**输入框聚焦：**

- 边框颜色变为霓虹青
- 外发光效果
- 标签上浮动画

**滚动指示器：**

- Hero 区域底部的向下箭头
- 轻微上下浮动（2s 循环）
- 点击平滑滚动到下一区域

**返回顶部按钮：**

- 滚动超过一屏后显示
- 点击平滑滚动到顶部
- 悬停时颜色变化

- 无结果提示

**关键实现：**

- data-type 属性标记
- CSS 过渡动画
- 数组过滤

### 6.5 模态框系统

**功能：**

- 项目详情展示
- 键盘支持（ESC 关闭）
- 点击背景关闭

**关键实现：**

- CSS 动画
- 键盘事件监听
- 焦点管理

## 七、响应式设计策略

### 7.1 移动端（< 768px）

- 单列布局
- 导航栏折叠为汉堡菜单
- 粒子数量减少（30-50）
- 字体大小缩小 10-15%
- 间距缩小 20-30%

### 7.2 平板（768px - 1024px）

- 双列布局
- 导航栏保持水平
- 粒子数量中等（50-100）
- 字体大小正常
- 间距正常

### 7.3 桌面（> 1024px）

- 三列布局
- 导航栏水平 + 主题切换按钮
- 粒子数量正常（100-150）
- 字体大小正常
- 间距正常

## 八、性能优化策略

### 8.1 资源优化

- **图片：** WebP 格式 + 懒加载 + 响应式图片
- **图标：** SVG 内联或 SVG Sprite
- **字体：** font-display: swap + 预加载关键字体
- **CSS：** 关键 CSS 内联 + 其余异步加载

### 8.2 JavaScript 优化

- **代码分割：** 按页面加载对应脚本
- **懒加载：** 非关键功能延迟加载
- **节流/防抖：** 滚动、resize 等事件
- **事件委托：** 减少事件监听器数量

### 8.3 渲染优化

- **硬件加速：** transform、opacity 动画
- **will-change：** 提前告知浏览器动画元素
- **避免强制同步布局：** 先读取后写入
- **虚拟滚动：** 项目列表（如果项目很多）

### 8.4 网络优化

- **预连接：** 提前建立连接
- **预加载：** 关键资源
- **缓存策略：** 静态资源长期缓存
- **CDN：** 加速资源加载

***

## 九、可访问性设计

### 9.1 语义化 HTML

- 使用正确的 HTML 标签
- 标题层级正确（h1 → h2 → h3）
- 列表使用 ul/ol
- 导航使用 nav

### 9.2 键盘支持

- 所有交互元素可通过 Tab 访问
- 焦点状态清晰可见
- ESC 关闭模态框
- 跳过导航链接

### 9.3 屏幕阅读器支持

- alt 属性描述图片
- aria-label 标注图标按钮
- aria-expanded 标注展开状态
- role 属性标注组件类型

### 9.4 颜色对比度

- 文字与背景对比度 ≥ 4.5:1
- 大文字对比度 ≥ 3:1
- 不仅依赖颜色传递信息

### 9.5 动画偏好

- 尊重 prefers-reduced-motion
- 用户禁用动画时，提供静态版本

## 十三、总结

本设计文档详细描述了一个极简科技风开发者个人主页的完整设计方案，包括：

1. **视觉系统：** 暗黑科技风配色、现代字体系统、统一间距
2. **页面设计：** 首页和项目页的详细布局和内容结构
3. **交互设计：** 粒子背景、滚动动画、悬停效果、主题切换
4. **技术实现：** 原生 HTML/CSS/JS，无外部依赖
5. **性能优化：** 资源优化、渲染优化、网络优化
6. **可访问性：** 语义化 HTML、键盘支持、屏幕阅读器支持
7. **SEO：** 元数据、结构化数据、语义化

设计遵循"极简/留白/高级感/轻交互/视觉统一"的核心原则，旨在打造一个既能展示开发者专业能力，又能体现个人品味的高级感个人主页。
