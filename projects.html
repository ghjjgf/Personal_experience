📅 2024年9月 — 2025年6月 后端开发工程师

基于计算机视觉的智能除草机

Python 华为昇腾 YOLO OpenCV

项目背景

该项目是校方与隶属于北大荒农垦集团有限公司的黑龙江省农垦科学院农业工程所的合作课题，主要针对北方大田作物环境下人工除杂成本高与自动化程度较低的问题，利用深度学习与端侧部署技术，精准识别秧苗与杂草，实现自动对行与精准除杂功能，解决苗间除草自动化程度较低与除杂精度较低等难题。

主要工作

数据增强

通过分析视频流中的视频帧构建初始图像数据集，并利用OpenCV框架实现自定义图像增强（曝光调整、高斯噪声、椒盐噪声、裁剪、旋转、边缘增强），提高数据集质量。

算法建模

在模型研发阶段，运用NVIDIA RTX A6000计算平台完成YOLO（v8、v10、v11）模型的多版本对比、训练与调优，同时自研的杂草中心像素点提取算法，显著提升了刀具操作精度。

端侧部署

在华为Atlas 200I DKA2开发板上成功自研实现YOLO11、YOLOv10、YOLOv8模型部署，并创新性地建立了端侧模型推理评价体系，解决在端侧设备中无法验证模型精度的问题。

项目成果

完成智能除草机视觉识别系统开发，实现秧苗与杂草识别与精准除草。在昇腾Atlas 200I DKA2端侧部署最优YOLO11模型，达到准确率85.3%、召回率75.9%、20FPS实时推理速度。

📅 2025年12月 — 2026年4月 核心成员

电能表时钟电池可靠性评价关键技术及在高严寒地区的应用研究

Python BPNN Qt

项目背景

该项目是校方与烟台东方威思顿有限公司的合作课题，主要针对电能表时钟电池的性能和可靠性问题，重点研究时钟电池钝化膜的快速消除、电池容量检测、时钟电池可靠性评价三大关键技术。

主要工作

数据增强

面向电池容量数据稀疏及小样本问题，设计基于多插值与LOOCV验证的安全数据增强方法，在保证数据分布合理性的前提下实现样本扩展，缓解过拟合并提升模型泛化能力。

特征提取

基于电池在不同容量阶段的电压响应曲线，挖掘关键特征参数并构建容量敏感特征集合，建立电压特性与容量演化之间的关联机制。

算法建模

构建基于BPNN的容量预测模型，实现容量检测任务的非线性建模与精确预测，支撑电池剩余容量检测与健康状态评估应用。

项目成果

完成电能表时钟电池剩余容量检测系统开发，并在Qt平台实现部署。BPNN模型指标为MAE=0.8507、MSE=1.2631、MAPE=1.34%，满足高寒地区电池可靠性评价需求，项目于2026年5月完成验收交付。

📅 2024年9月 — 2026年6月 负责人

工地智慧安全态势感知与监测平台

Python YOLO Qt C++ OpenVINO

项目背景

项目旨在开发一种先进的低成本的无人机监控系统，以应对施工现场的多重安全挑战，例如火灾、人员着装不达标等，同时降低无人机的适配成本，使小型无人机也可完成监测任务。

主要工作

数据构建

面向多类别异常检测任务，构建多源开源数据集并完成数据清洗与重构，按照标准比例划分训练集、验证集与测试集，形成规范化模型训练数据体系。

模型部署

基于Qt构建可视化推理系统，支持视频流输入与检测结果动态展示，实现异常的实时加载与检测调用。

模型加速

基于OpenVINO加速库优化模型在CPU环境下的推理速度，满足实际应用场景的实时性需求。

硬件交互

系统监测到安全隐患后，与树莓派通信并同时发送警报信号给蜂鸣器，完成从监测到预警响应效果。

项目成果

完成智慧工地安全监测平台研发，实现异常识别、实时预警与设备联动。YOLO模型指标为准确率88.0%、召回率63.6%，实现监测-预警-处置-反馈全链路开发，项目于2026年6月结题。
